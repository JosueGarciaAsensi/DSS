# DSS Project

<br>

## Inicialización del proyecto

Se facilita el archivo ```.env``` en el índice privado. Una vez copiado en el directorio de Laravel, se deben ejecutar los siguientes comandos.
```
composer install --no-scripts
composer dump-load
```
